start
-----
{% highlight bash %}
mongo
{% endhighlight %}

***
Connect to database
-------------------
{% highlight bash %}
show dbs
use local
{% endhighlight %}

***
See database tables
-------------------
{% highlight mysql %}
show collections
db.funnytable.find()
{% endhighlight %}

***
Regex
-----
{% highlight mysql %}
db.funnytable.find(
  $or: [
        {'name': { $regex: req.params.username, $options: 'i' }},
        {'username': { $regex: req.params.username, $options: 'i' }},
        {'email': { $regex: req.params.username, $options: 'i' }}
       ])
{% endhighlight %}
